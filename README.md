## Cheat Sheet

### Variables & Datatypes

**lists** are mutable, meaning to can change them
```
list = ['one', 'two', 'three']
```

**tuples** are immutable, meaning they do not change
```
tuple = (1,2,3)
```

**dict** or dictionaries are a key-value pairs
```
dict = {Key1: "Value1", Key2: "Value2"}
```

**here string** 
```
string = """
	some string
"""
```

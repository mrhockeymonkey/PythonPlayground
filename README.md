# Cheat Sheet

**list** a list is mutable, meaning to can change them
```
list = ['one', 'two', 'three']
```

**tuples** tuples are immutable, meaning they do not change
```
tuple = (1,2,3)
```

**dict** dictionaries are a key-value pairs or hashtables
```
dict = {Key1: "Value1", Key2: "Value2"}
```
